# Analyse des disparités de revenus en Suisse

## Objectif
Analyser les facteurs pouvant expliquer les disparités de revenus en Suisse selon les régions, l’âge, le niveau d’éducation et le type de ménage.

## Compétences utilisées
- Analyse exploratoire de données
- Visualisation de données
- Régression linéaire
- Interprétation statistique
- Data storytelling

## Travail réalisé
- Analyse du revenu moyen par région
- Comparaison entre le niveau d’éducation et le revenu
- Étude de la relation entre l’âge et le revenu
- Analyse de l’influence du type de ménage sur les revenus
- Création de visualisations pour présenter les résultats

## Résultats
- Les revenus varient fortement selon les régions.
- L’âge et le niveau d’éducation ne suffisent pas à expliquer les écarts observés.
- Le type de ménage semble jouer un rôle important dans les différences de revenus.

## Technologies
- Python
- Pandas
- Matplotlib
