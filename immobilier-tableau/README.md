# Analyse de données & recommandations – Investissement immobilier

## Objectif
Analyser des données de locations pour recommander les meilleures villes d’investissement.

## Données analysées
- Villes : Winnipeg (Canada), La Haye (Pays-Bas)
- Types de voyageurs (business, famille)
- Saisonnalité des réservations

## Outils
- Tableau

## Résultats
- La Haye : forte demande de voyageurs d’affaires durant l’année
- Winnipeg : forte demande familiale durant l’été

## Recommandation
- Investir à La Haye pour une rentabilité stable annuelle
- Investir à Winnipeg pour maximiser les revenus saisonniers
